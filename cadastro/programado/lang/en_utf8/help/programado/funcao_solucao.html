Função de solução

Neste campo deverá ser digitado uma função compatível com a linguagem PHP. Segue um exemplo:

	
	function correcao($resposta, $arg1) {

	  $teste = base_convert($arg1 , 10, 16);

	  if ($teste == $resposta) 
	  {
	    return 1;
	  }

	  return 0;
	}

Os parâmetros de entrada da função serão digitados logo abaixo, para isso você deve clicar no botão 'adicionar novo parâmetro'.
